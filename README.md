# PDFViewerInteractiveMagazine
Сайт будет представлять из себя сборники статей в PDF-формате на разрные тематики. К статьям читатели смогут добавлять аннотации, выделяя их цветом прямо в тексте. К аннотациям будет возможно написать комментарии - продолжить ветку обсуждения или начать новую.
Вьювер реализуется на базе pdf.js
